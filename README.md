# Staybnb 

An Airbnb-inspired responsive frontend website built using HTML, CSS, and JavaScript.

## Features

- Responsive design
- Multi-page layout
- Property listing interface
- House details page
- Navigation components
- Modern UI inspired by Airbnb
- Mobile-friendly design

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Project Structure

```text
staybnb-frontend_project/
│
├── index.html
├── listing.html
├── house.html
├── css/ style.css
├── images/
├── screenshots/
└── README.md
```
